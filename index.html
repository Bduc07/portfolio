Weather App

Project Description:

Developed a weather application that fetches and displays real-time weather data, including current temperature, weather conditions, wind speed, and humidity for a specified city. The app also displays the current date and time.

Technologies Used:

  • HTML/CSS: Structured the web page layout and styling.
  • JavaScript: Implemented functionality to fetch and display weather data using the OpenWeatherMap API.
  • API Integration: Utilized the OpenWeatherMap API to retrieve weather information.
  • Date and Time Handling: Displayed current date and time with JavaScript Date object.

Key Features:

  • Dynamic Data Fetching: Used fetch API to retrieve and display real-time weather data based on user input.
  • Weather Data Display: Showed current temperature, maximum and minimum temperatures, feels-like temperature, humidity, and wind speed.

  • Date and Time Display: Added functionality to display the current date and time.

Responsibilities:

  • Designed and implemented the user interface for displaying weather information.

Caesar Cipher Program

Project Overview:

I developed a Caesar Cipher program in Python to encrypt and decrypt messages. The Caesar Cipher is a basic encryption technique where each letter in the message is shifted a certain number of places in the alphabet.

What I Did:

  • User Interaction:
    • Created a user-friendly interface that asks whether the user wants to encrypt or decrypt a message or file.
    • Implemented input prompts to get the message, shift value, and file names from the user.
  • Encryption and Decryption:
    • Wrote functions to shift letters in the message by a specified number of places to encrypt it.
    • Developed a function to reverse the process and decrypt the message.
  • File Handling:
    • Added functionality to read text from files, encrypt or decrypt it, and save the results back to a new file.
    • Checked if the specified file exists before trying to read it.

Skills Demonstrated:

  • Python Programming: Used Python to build the entire program and handle various tasks.
  • Text Processing: Applied algorithms to manipulate and encode/decode text.
  • File Operations: Managed reading from and writing to files efficiently.
  • User Interaction: Designed a simple interface for users to interact with the program.
